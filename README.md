# crop_recommendation_yield_prediction_app
AI-based crop recommendation and yield prediction system built using Streamlit and machine learning. It analyzes soil nutrients and weather conditions to suggest suitable crops and estimate yield, helping farmers make better agricultural decisions.
Technologies:
- Python
- Streamlit
- Scikit-learn
- Pandas

Models used:
- Random Forest Classifier
- Random Forest Regressor

Features:
- Crop recommendation
- Yield prediction
- Streamlit web interface
